Welcome to the Restaurant Recommender System! 🎉

This system implements four different recommendation algorithms as part of my BSP 3 project:

  • Content-Based Filtering: Uses cosine similarity to find restaurants with similar features
  • Popularity-Based Filtering: Recommends highly-rated restaurants using weighted scoring (50% rating, 30% sentiment, 20% reviews)
  • Category-Based Filtering: Groups restaurants by cuisine categories
  • K-Means Clustering: Uses machine learning with 7 clusters to find similar restaurants

Select an algorithm above and enter a restaurant name to get started!
