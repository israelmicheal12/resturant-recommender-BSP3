# Web-Based Restaurant Recommender System (BSP3)

This project implements a web-based restaurant recommender system using Flask and machine learning techniques.

## Features
- Search restaurants by name
- Filter by city, cuisine, price range, and rating
- Content-based recommendations using cosine similarity
- Popularity-based recommendations
- K-Means clustering
- Sentiment analysis using TextBlob

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Pandas, NumPy
- Scikit-learn
- TextBlob, FuzzyWuzzy

## Deployment
The application is deployed using GitHub and Render and can be accessed through a web browser.
